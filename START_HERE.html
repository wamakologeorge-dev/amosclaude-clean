AAmosclaud Agent Server
PORTABLE · SELF-HOSTED · FOLDER-FIRST

Your agent platform lives here.

This folder contains the application, server, workspace, repositories, model-provider configuration, and persistent data layout.

WINDOWS

Start the application

Double-click start-windows.bat. Amosclaud prepares its private Python environment on the first run.

LINUX · macOS

Start from Terminal

Open this folder in Terminal and run:

chmod +x start-local.sh && ./start-local.sh
OPEN

Use the workspace

After the server starts, open http://localhost:8000 in your browser.

CONFIGURE

Connect the agent model

Purchase Amosclaud agent credits, create an installation key, then place AMOSCLAUD_API_KEY in .env. Owner provider secrets are never distributed.

Keep the data folder backed up. It contains your local Amosclaud state.
