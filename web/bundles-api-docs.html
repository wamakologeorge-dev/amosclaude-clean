API reference

Base path: /api/v1. Protected operations accept Bearer authentication or X-API-Key.

GET/bundles/health
GET/bundles
POST/bundles
GET/bundles/{bundle_id}
GET/bundles/{bundle_id}/download

Catalog request

curl -H "Authorization: Bearer $AMOSCLAUD_BUNDLES_API_KEY" \
  https://amosclaud.com/api/v1/bundles

Upload request

curl -X POST https://amosclaud.com/api/v1/bundles \
  -H "X-API-Key: $AMOSCLAUD_BUNDLES_API_KEY" \
  -F version=1.0.0 -F platform=linux-x86_64 \
  -F channel=stable -F file=@Amosclaud-Server.tar.gz

Verify the X-Amosclaud-SHA256 response header after every download.
