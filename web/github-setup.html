GitHub Integration Setup

This page prevents the raw “GitHub integration is not configured” screen and shows exactly what the runtime still needs.

Checking GitHub integration…

Required Railway variables

GITHUB_CLIENT_ID=your GitHub OAuth App client ID GITHUB_CLIENT_SECRET=your GitHub OAuth App client secret GITHUB_TOKEN_ENCRYPTION_KEY=a long random private secret GITHUB_REPOSITORY_CALLBACK_URL=http://www.amosclaud.com/api/v1/github/callback

Store these only in Railway Variables. Never place the client secret or encryption key in GitHub source code.
