Amosclaud operators

The old public collection of autonomous agents is replaced by two governed platform operators. Amosclaud-bot receives and coordinates work. Amosclaud Fixer diagnoses, repairs, tests, and returns verifiable evidence.

Main operator

Amosclaud-bot

Connected to the real command behavior demonstrated by Amosclaud-agent issue #6. It accepts project and issue objectives, creates one central task, and routes work through the existing GitHub App and execution system.

  • Create and inspect real projects
  • Create and manage issues
  • Coordinate build, review, test, deploy, and monitor work
  • Return one shared task history and result

Repair operator

Amosclaud Fixer

The trusted repair specialist under Amosclaud-bot. It runs evidence-first diagnosis, applies bounded repairs, executes tests, rolls back unverifiable changes, and publishes real commits or pull requests only when verification succeeds.

  • Diagnose workflow and repository failures
  • Apply safe deterministic repairs
  • Run tests and Doctor verification
  • Publish proof, logs, commits, and pull requests
