Install Amosclaud Server

Download the organized server workspace, verify its checksum, extract it, and run the installer for your operating system.

Download Windows ZIP

Verify SHA-256 checksums

  1. Extract the Amosclaud folder.
  2. Run install-amosclaud.ps1 on Windows or ./install-amosclaud.sh on Linux/macOS.
  3. Open http://localhost:8000 after the health check passes.
