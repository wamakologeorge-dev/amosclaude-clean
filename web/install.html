Install Amosclaud on Windows

Amosclaud runs the dashboard, backend, agent, local model, Vault, database, repositories, and storage on one computer.

1. Download

Download the latest verified Windows package.

Download Amosclaud

2. Extract

Right-click the ZIP and select Extract All. Do not run it from inside the compressed-file preview.

3. Install Docker Desktop

Install and start Docker Desktop. Wait until its engine reports that it is running.

4. Configure ownership

Double-click start-amosclaud.bat. On first start it creates .env. Replace owner@example.com with your email before exposing the installation to anyone else.

5. Open Amosclaud

The launcher starts the local model and opens Chrome at http://localhost:8000. The first model download can take several minutes.

Verify the package

Every release also includes Amosclaud-Windows.zip.sha256. You can compare its SHA-256 value with the downloaded ZIP before installation.
