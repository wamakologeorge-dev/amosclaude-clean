# Amosclaud OS

Amosclaud OS is the native engineering backend for **Amosclaud.com**.

Its permanent mission is to let a signed-in user create and operate workspaces, repositories, files, branches, issues, CI jobs, releases, and deployments without leaving Amosclaud.com.

The visible operator is **Amosclaud Agent**. Internally it coordinates Amosclaud-bot, Codex-style planning, Amosclaud Autonomous execution, Amosclaud Doctor diagnosis, Amosclaud Fixer repair, BIG CI, and verification.

The runtime Python package is named `amosclaud_os` because Python import names cannot contain hyphens.
