Pull request launcher

Open a GitHub pull request from the Agent Issue Hub.

Choose the repository and branches. The page sends you to GitHub’s official compare and pull-request screen, where GitHub checks your identity and permission before anything is created.

The branches must already exist on GitHub. This page does not create branches, commits, or pull requests directly.

Secure PR path

GitHub remains in control

  • The page builds only a GitHub compare URL.
  • GitHub requires the user to sign in.
  • GitHub verifies repository access and branch permissions.
  • The user reviews the diff before creating the PR.
  • Amosclaud checks and Doctor verification can run after the PR opens.

No GitHub token is collected or stored. No PR is merged automatically.

Limited-role security

Everyone can participate. Authority stays with GitHub.

The page does not invent a new account system. GitHub identity and repository permissions remain the source of truth.
