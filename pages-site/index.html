Pull request launcher

Prepare repository work from the website.

Choose the repository and branches. GitHub remains the source of truth and verifies identity and permissions before the record is created.

The branches must already exist on GitHub. This page does not create branches, commits, or pull requests directly.

Secure PR path

GitHub remains the authority

  • The website prepares a governed repository action.
  • GitHub verifies identity and repository permissions.
  • The user reviews the diff before creating the PR.
  • Amosclaud checks and Doctor verification run after the PR opens.
  • The website displays the PR and its evidence.

No GitHub token is collected or stored. No PR is merged automatically.

Limited-role security

Everyone can participate. Authority stays with GitHub.

The page does not invent a new account system. GitHub identity and repository permissions remain the source of truth.
