# Amosclaud Bot commands

```text
@amosclaud status
@amosclaud inspect <objective>
@amosclaud review this PR
@amosclaud verify <objective>
@amosclaud fix <specific problem>
```

`review` is read-only and produces the professional pull-request review report.

`fix` is the write-capable repair path and remains restricted to trusted repository collaborators.
